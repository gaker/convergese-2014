Install Virtualenv
==================

    $ sudo pip install virtualenv


<br><br>


<div class="fragment">
    <h2>Create virtualenv</h2>

    <pre><code data-trim>
    $ virtualenv env
    </code></pre>
</div>

<br><br>

<div class="fragment">
    <h2>"Activate" the virtualenv</h2>

    <pre><code data-trim>
    $ source env/bin/activate
    </code></pre>
</div>
